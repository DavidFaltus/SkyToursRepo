# SkyTours - Rezervační systém letů

## Pokyny pro spuštění

### 1. Spuštění databáze přes Docker:
Pro spuštění instance databáze stačí spustit soubor **`docker-compose.yml`**:

### 2. Spuštění aplikace Přes IDE:
Hlavní třída **`SpolecnyProjektUhkApplication`**.

Aplikace je následně dostupná na: [http://localhost:8080](http://localhost:8080)

### 3. Spuštění Unit testů
Pro spuštění všech testů stačí spustit soubor **`run-tests.ps`**:, který se nachází ve složce **main/resources/static/tests**
